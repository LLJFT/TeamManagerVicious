# Vicious Explainer — Voiceover Recording Brief

**For:** VO talent + recording engineer
**Pairs with:** `Explainer_Video_Production.md` (locked script in §3), `assets/vo-long.wav` and `assets/vo-short.wav` (timing reference — TTS scratch only)
**Deliverable back to producer:** Dry, mastered WAVs (long + short), plus raw session and alt takes.

---

## 1. Brand voice in one paragraph

Vicious is a tactical command center for serious esports orgs. The read is a head coach at halftime — confident, quietly intense, never hyped. Think *Athletic Greens narrator meets a tactical analyst*. Lower‑mid register, dry American or neutral mid‑Atlantic, ~150 wpm, no upspeak, no smile in the voice, no "announcer" energy. We are not selling — we are stating facts the listener already half‑believes.

---

## 2. Talent profile

- **Gender:** Open (male or female). Pick the take that lands flatter and more authoritative.
- **Age range:** 30–45.
- **Accent:** General American or neutral mid‑Atlantic. No regional inflection.
- **Tone references:** Apple product film narrator (restrained), Linear changelog VO, ESPN feature‑doc narration, *Drive to Survive* analyst beats.
- **Avoid:** "Movie trailer guy," radio‑DJ warmth, hype‑video cadence, vocal fry on tails.
- **Royalty‑free fallback (for scratch only, not master):** ElevenLabs Adam or Rachel, stability 0.5 / similarity 0.75.

---

## 3. Session technical spec

| Item | Spec |
|---|---|
| Sample rate / bit depth | 48 kHz / 24‑bit |
| Format | Mono WAV, uncompressed |
| Mic | Large‑diaphragm condenser (Neumann TLM 103, U87, Sennheiser MKH 416, or equivalent) |
| Distance | 6–8 in, slight off‑axis to reduce plosives |
| Booth | Treated, RT60 < 0.25s, noise floor ≤ −60 dBFS |
| Headphone monitoring | Closed‑back, click track + scratch VO bed at −12 dB |
| Levels | Peaks −6 dBFS, average −18 to −20 dBFS RMS, no limiting on the way in |
| Edits | Dry — no reverb, EQ, comp, de‑ess, or noise gate baked in |
| File naming | `vicious_vo_<cut>_<line#>_<takeN>.wav` (e.g. `vicious_vo_long_03_take2.wav`) |

## 4. The script (locked — read verbatim)

> **Source of truth:** `Explainer_Video_Production.md` §3. The lines below are reproduced for the booth. **Do not paraphrase, drop articles, or "improve" copy.** Italics in parentheses are direction, not read.

| # | TC | Line | Direction |
|--:|---|---|---|

| 2 | 0:08 | "And nobody knows who's actually playing." | Drop a half‑step. Resigned, not sarcastic. |
| 3 | 0:14 | "This is how seasons quietly fall apart." | Slowest line in the piece. Land "quietly" with weight. |
| 4 | 0:20 | "Vicious is the command center for serious esports orgs." | Pivot point. Sit up. Confident, declarative — but do **not** smile. |
| 5 | 0:28 | "One roster. Every game. Every match." | Three crisp beats. Equal weight on all three. |
| 6 | 0:34 | "Scout the opponent. Lock the draft. Run the comp." | Tactical cadence. Each verb gets a hit. |

| 8 | 0:54 | "Scale across teams, titles, and staff. One subscription. One source of truth." | Settle. This is the org‑level promise. |
| 9 | 1:02 | "Less spreadsheet. More winning." | Dry. Almost throwaway. The line lands because you don't sell it. |
| 10 | 1:08 | "Vicious. Run your roster like a pro." | Tag line. Confident period. |

### Pickup / alt lines (record at end of session)

- "Vicious. Run your roster like a pro." — 3 alt reads: (a) drier, (b) slightly warmer, (c) half‑beat slower.
- "vicious dot gee gee" — 2 alt reads: (a) flat, (b) with a quarter‑beat lift on "gg."
- "Less spreadsheet. More winning." — 1 alt with a beat between the two sentences for use in the short cut if pacing tightens.

## 5. Take strategy

- **3 full passes** of each cut, top‑to‑bottom, no stops, before line‑by‑line work. Producer picks the spine pass; line pickups fill the rest.
- **Per line:** 2 on‑direction takes + 1 "throwaway" — flatter, almost bored. The throwaways frequently win in this brand voice.
- **Slate every take** verbally: "Vicious, long cut, line three, take two."
- Do **not** match the scratch TTS rhythm beat‑for‑beat. The scratch is a duration target, not a performance reference. If a line lands tighter or longer, mark it and we'll re‑time picture.

---

## 6. Pronunciation & emphasis notes

| Word / phrase | Notes |
|---|---|
| Vicious | "VISH‑us." Two syllables, equal weight. Never "VISH‑ee‑us." |
| vicious.gg | "vicious dot gee gee." Both Gs pronounced as letters, not as a word. |
| esports | One word, no hyphen, stress on "E." Not "ee‑sports." |
| scrim | Hard "k." Short "i." (Rhymes with "trim.") |
| comp | Short. Not "compositions" or "comps." |
| draft | American "a." (Not British "draahft.") |
| roster | American "ROS‑ter," not "ROW‑ster." |

## 7. Delivery package back to producer

Deliver via the producer's drop folder within 24h of session:

1. `vicious_vo_long_master.wav` — selected master, dry, head/tail trimmed, peaks at −6 dBFS.
2. `vicious_vo_short_master.wav` — same.
3. `/raw/` — every numbered take, untrimmed, with slate.
4. `/alts/` — pickup and alt reads from §4.
5. `notes.txt` — any line where talent flagged a different rhythm than the script's timecodes, plus any pronunciation calls the engineer made on the floor.

---

## 8. Approval loop

- Producer reviews within 1 business day, returns either **APPROVED** or a single consolidated revision list (max one round expected).
- Revisions ship same‑day from talent.
- Final masters drop into `deliverables/explainer_video/assets/` as `vo-long.wav` and `vo-short.wav`, replacing the current TTS scratch files. The motion editor pulls from those filenames — do not rename.
