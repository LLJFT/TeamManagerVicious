User roles & permissions:
	∙	Super Admin (me) — sees everything across all games

	∙	Game Manager — can be assigned to one or multiple games, manages those games only
	∙	Coach/Analyst — sees their game only
	∙	Player — sees their game only

	1.	User goes to vicious.services
	2.	Clicks Register — enters: username + selects game(s) + selects role
	3.	Sees waiting page: “Your request is under review”
	4.	Request appears in TWO places simultaneously:
	∙	Org Admin dashboard (home)
	∙	Game Manager dashboard (for that specific game)
	5.	Either one can approve or reject
	6.	On approval → user gets in-app notification (no email needed, no email required at registration)
	7.	User can now access their game page(s)
Home page (after login):
Shows all 24 games as cards. Games the user has access to = clickable. Others = locked 🔒
Org Admin home dashboard shows:
	1.	Attendance summary per game (Attended / Late / Absent from actual events — NOT availability)
	2.	Recent results per game (Win/Loss)
	3.	All users list with their game and role
	4.	Pending registration requests
